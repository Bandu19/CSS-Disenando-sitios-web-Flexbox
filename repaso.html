Propiedades Nuevas de Flex del curso


Display Flex

Para display flex se nececita de dos cosas, una es la propiedad de display:flex y la otra es mas que nada el flex-direction: row;
flex-direction:row tiene mas de una propiedad como colum o row-reverse, se puede probar de diferente manera con tal de saber su uso, en la imagen se muestra un ejemplo de como se utiliza la propiedad de flex con flex-direction: row;(fila)

Flex-direction:colum;

Para la propiedad colum se puede observar en la siguiente imagen donde muestra el ejemplo.

Flex-wrap: wrap;

wrap es mas que nada que junto con el row hace que las cajas no pierdan su ancho y este crea otra fila dependiendo del contenedor

Exite otro elemento adaptable que reduce un poco el codigo, la forma de reducirlo es agregando la propiedad flex-flow: row wrap; que en pocas palabras significa agregar flex-direction: row; con flex-wrap: wrap;
